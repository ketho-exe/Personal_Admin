# Personal Admin

## Database setup

Run the SQL in [`supabase/schema.sql`](./supabase/schema.sql) in your Supabase project's SQL editor to create the schema, triggers, policies, and helper view required by this app. In short: run it in your Supabase project before using the app.

1. Open your Supabase project.
2. Go to `SQL Editor`.
3. Paste in the contents of `supabase/schema.sql` and run it.

Re-run the same file in Supabase whenever you need to recreate the database structure from source.
